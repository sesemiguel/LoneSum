# Lone Sum

##### Given Problem
Given 3 int values, a b c, return their sum. However, if one of the values is the same as another of the values, it does not count towards the sum.

##### Sample Input
loneSum(1,2,3)

loneSum(3,2,3)

loneSum(3,3,3)


##### Sample Output
6

2

0
